## 💡 솔루션 (3분)

**[슬라이드 5: 404 Found 로고]**

> 저희는 404조입니다.  
> 개발자에게 404는 "Not Found", 즉 찾을 수 없다는 뜻이죠.  
> 하지만 저희는 **"404 Found"**를 만들었습니다.  
> **응급 환자를 위한 병원을, AI가 찾아드립니다.**

**[슬라이드 6: 서비스 플로우]**

**1단계: 응급대원이 체크리스트 입력**

- 환자 나이, 성별
- 증상 (의식 수준, 출혈, 통증)
- KTAS 중증도 (1~5등급)
- 특이사항

**2단계: AI가 자동으로 병원에 전화**

- Twilio 기반 자동 발신
- 여러 병원에 **동시에** 전화
- 거리순/중증도 우선순위 자동 계산 (C++)

**3단계: AI 음성 대화로 병원 승인**

- "응급 환자 발생. 27세 남성, KTAS 2등급, 복부 출혈..."
- 병원: "지금 응급실 상황이..."
- AI: "알겠습니다. 다른 병원 연락드리겠습니다."

**4단계: 승인 즉시 다른 병원 자동 종료**

- 한 병원이 승인하면 나머지 전화 자동 끊김
- 응급대원에게 실시간 알림
- **골든타임 최대 50% 단축**

**[슬라이드 7: 기술 스택]**

> 그럼 어떻게 만들었을까요?

**Frontend (Vercel 배포 완료)**

- Next.js 16 + TypeScript
- 직관적인 체크리스트 UI
- 실시간 병원 응답 현황

**Backend (AWS 배포 중)**

- NestJS (메인 API 서버)
- Python FastAPI (AI 음성 처리)
- C++ (병원 거리 계산 - 고성능)
- Docker Compose 통합

**AI & Voice**

- Twilio (국제 음성 통화)
- Google Cloud TTS/STT (한국어)
- GPT-4o (자연스러운 대화)

## 📝 Q&A 예상 질문 대비

**Q1: 실제로 병원에서 사용할 수 있나요?**

> A: 현재는 프로토타입이지만, 실제 병원 협업을 통해 파일럿 테스트 진행 예정입니다. 의료법 규제와 개인정보 보호는 HIPAA 기준으로 설계했습니다.

**Q2: AI가 오판할 위험은 없나요?**

> A: 최종 결정은 항상 **병원 의료진**이 합니다. AI는 정보 전달만 담당하고, 승인/거절은 사람이 판단합니다.

**Q3: 비용은 얼마나 드나요?**

> A: Twilio 통화료는 분당 약 50원, GPT-4o API는 요청당 10원 수준입니다. 한 건당 총 100~200원 이내로 매우 저렴합니다.

**Q4: 한국어 음성 품질은?**

> A: Google Cloud TTS Wavenet을 사용해 자연스러운 한국어 음성을 제공합니다. 데모 들어보시겠습니까?

**Q5: 다른 나라에서도 사용 가능한가요?**

> A: Twilio는 190개국 지원, GPT-4o는 다국어 지원으로 글로벌 확장 가능합니다.
